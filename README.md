# Helio Babel Presets

> Sets up node projects with proper ES2017+ sytax

## Installation

Using NPM

```bash
$ npm install @helio-training/babel-preset-helio -D
```

Using Yarn

```bash
$ yarn add @helio-training/babel-preset-helio --dev
```

### Usage

`.babelrc` Preferred

```json
{
  "presets": "helio"
}
```
